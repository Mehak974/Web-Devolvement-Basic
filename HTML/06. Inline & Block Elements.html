This is an inline element. It will only take space necessary for the contant within the tag and start right after the previous tag finishes.

Inline elements in HTML:

a, b, em, br, i, img, input, button, label, q, select, span, strong, sub, sup, textarea, abbr, acronym, bdo, big, cite, code, dfn, kbd, map, object, output, samp, script, small, time, tt, var
